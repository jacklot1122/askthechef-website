# Ask The Chef - Private Chef Website

A modern, responsive website for a private chef service in Brisbane, Australia.

## Features

- Restaurant-quality private chef experiences
- Weekly in-home chef services
- Family & lifestyle meal preparation
- Small gatherings (up to 12 guests)

## Tech Stack

- Vanilla HTML5, CSS3, JavaScript
- Google Fonts (DM Serif Display, DM Sans)
- Fully responsive design
- No build step required

## Deployment

This project is Vercel-ready. Simply:

1. Connect your GitHub repository to Vercel
2. Deploy with zero configuration

Or deploy manually:

```bash
vercel
```

## Local Development

```bash
# Using Python
python -m http.server 8080

# Using Node.js
npx serve
```

Then open http://localhost:8080
